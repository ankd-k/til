# create new gatsby site

```shell
$ npm i -g gatsby # グローバルにgatsbyをインストール（＝gatsby-cliをインストール）
$ gatsby new # 新しくサイトを作成
```

ここから以下の内容を聞かれるので答える

1. サイト名
2. フォルダ名
3. 使用言語（js or ts）
4. 使用するCMS
5. 使用するCSSフレームワーク
6. 追加するプラグイン
7. 1~6の内容確認

```shell
$ cd 2のフォルダ名
$ gatsby develop # ≒ npm run develop
```
